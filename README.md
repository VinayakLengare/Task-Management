# Task Management 

• Built a full-stack Task Management system using Spring Boot for backend and React.js for frontend. 
• Developed RESTful APIs using Spring Boot to handle task creation, update, deletion, and status tracking. 
• Implemented Spring Data JPA with MySQL for efficient database operations and entity relationships. 
• Applied JWT-based authentication to secure APIs and manage user access. 
• Used proper exception handling and validation to ensure reliable backend performance.
